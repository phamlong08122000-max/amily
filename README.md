# Amilly Carter - Bio Link Page

Deploy tự động lên Vercel.

## Cách deploy:

1. Upload folder này lên GitHub (repo mới)
2. Vào [vercel.com](https://vercel.com) → Import Git Repository
3. Chọn repo → Deploy
4. Xong! Vercel tự detect static site

## Files:
- `index.html` — trang chính
- `style.css` — CSS
- `vercel.json` — cấu hình Vercel
